PdcVipSolutions

Version 2.0

The PdcVipSolutions v.2.0 - is the set of tools (as the additions to PDC Visual Prolog official product), examples (tutorial and demonstrational) and projects to build useful applications.
The main purpose of version 2.0 is the demonstration of the WorkSpace Manager Tool as the executable application and its source code.

The current  version 2.0 is build by Vip version 9x (in fact 9006). 
The full content is placed at https://github.com/PdcVipSolutions

The content

All content must be placed to directories with names
SpbRSolutions (the root directory may have any other name)
    SpbVipTools (the directory is created by download)
    SpbExamples (the directory is created by download)
    SpbProjects  (the directory must be created manually)

SpbVipTools
    SpbVipTools is the set of source code (Vip-packages) to support project and  examples and to demonstrate how the WorkSpace Manager works.
    SpbVipTools also contains the ready to use executables of Applications, which are represented at SpbProjects directory.
    You do not need to have VIP9x to run executables. Just run it!
    To get SPB VipTools you must download it as one bunch from https://github.com/PdcVipSolutions/SpbVipTools

Executables and the documentation

The executables of applications are placed at SpbRSolutions/SpbVipTools/Bin.
The documentation for the WorksSpace Manager placed at SpbRSolutions/SpbVipTools/Bin/wsmAppData, where you can find
    ReadMe(En/Ru).pdf - the instruction how to build and run applications
    WorkSpaceManager(En/Ru).pdf - the general Info regarding the tool
    wsm_help(En/Ru).chm    -  the standard MS Help, which also may be called from the WorkSpaceManager.exe application.

WorkSpaceManager also is represented as the integration of http-based backend and http-based frontend. This is the first step toward the applications based on the microservice ideas.
    It has NO mullty-user functionality supported.
    It has NO reaction on the not started backend.
The main purpose was to get the list of tasks to be solved in the future.
