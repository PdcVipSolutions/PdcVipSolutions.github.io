PdcVipSolutions

Version 2.0

PdcVipSolutions v.2.0 - это набор инструментов (как дополнение к официальному продукту PDC Visual Prolog), примеров (учебных и демонстрационных) и проектов для создания полезных приложений.
Основная цель версии 2.0 - демонстрация инструмента WorkSpace Manager как исполняемого приложения и его исходных кодов.
Текущая версия 2.0 построена на Vip версии 9x (фактически 9006).
Полный контент размещен по адресу https://github.com/PdcVipSolutions.

Содержание

Весь контент должен быть размещен в каталогах с именами
SpbRSolutions (корневой каталог может иметь любое другое имя)
     SpbVipTools (директория создается в результате загрузки)
     SpbExamples (директория создается в результате загрузки)
     SpbProjects (директория должна быть создана вручную)

SpbVipTools
     SpbVipTools - это пакеты исходного кода (Vip-пакеты) для поддержки проектов и примеров, а также для демонстрации работы WorkSpace Manager.
     SpbVipTools также содержит готовые к использованию исполняемые файлы Приложений, которые представлены в каталоге SpbProjects.
     Вам не нужно иметь VIP9x для запуска исполняемых файлов. Просто запустите это!
     Чтобы получить SPB VipTools, вы должны загрузить его одной связкой с https://github.com/PdcVipSolutions/SpbVipTools
     Структура контента показана ниже



SpbExamples
    Чтобы получить содержимое SpbExamples, вы должны загрузить его как однрй загрузкой с https://github.com/PdcVipSolutions/SpbExamples
    Структура контента показана ниже

    Примеры, которые входят в версию 2.0, были созданы много лет назад. Они обновлены только до состояния инструментария VIP9x.


SpbProjects
    Содержит отдельно загружаемые проекты:
          WorkSpaceManager (загрузить с https://github.com/PdcVipSolutions/WS_Manager)
          DelFiles (загрузить с https://github.com/PdcVipSolutions/DelFiles)
     Структура контента представлена ниже


Исполняемые файлы и документация

Исполняемые файлы приложений размещаются в SpbRSolutions\SpbVipTools\Bin.
Документация для WorksSpace Manager размещена по адресу SpbRSolutions\SpbVipTools\Bin\wsmAppData, где вы можете найти
     ReadMe(En/Ru).pdf - инструкция по сборке и запуску приложений
     WorkSpaceManager(En/Ru).pdf - общая информация об инструменте
     wsm_help(En/Ru).chm - стандартная MS-справка, которую также можно вызвать из приложения WorkSpaceManager.exe.

WorkSpaceManager также представлен как интеграция бэкэнда на основе http и фронтэнда на основе http. Это первый шаг к приложениям, основанным на идеях микросервиса.
     У него НЕТ функциональности, поддерживающей многопользовательский режим.
     Он не предстматривает проверки запущен ли  бэкэнд.
Основной целью было получить список задач, которые предстоит решить в будущем.
